# اسم المشروع: Wuzzuf Scraping Tool 🚀

### وصف بسيط:
كود بايثون احترافي لسحب وظائف البرمجة من موقع **Wuzzuf** وتخزينها في ملف CSV (يفتح بواسطة Excel).

### المكتبات المستخدمة:
* `requests`
* `BeautifulSoup`
* `csv`
* `itertools`

### المميزات التقنية:
* 🛡️ **Error Handling:** الكود لا يتوقف إذا واجه بيانات ناقصة بفضل استخدام جمل `try/except`.
* 📄 **Pagination:** الكود ينتقل بين الصفحات تلقائياً لسحب أكبر قدر من الوظائف.
* 🔍 **Detailed Scraping:** الكود لا يكتفي بالصفحة الرئيسية، بل يدخل لروابط الوظائف لجلب تفاصيل دقيقة مثل **الرواتب** و**المسؤوليات**.
